

# 📘 AutoGen中文翻译文档

## 🚀 关于AutoGen
AutoGen是一个由[Microsoft](https://github.com/microsoft)开源的框架，专为构建和优化大型语言模型（LLM）工作流程而设计。它提供了多代理会话框架、应用程序构建工具以及推理性能优化的支持。

## 🌐 初心和目标
